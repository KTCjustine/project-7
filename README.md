# Tailwind-Boilerplate
Tailwind-Boilerplate for development

### Usage Guide
1. Run npm install after cloning the repository
2. Update page title
3. Check the fonts if it is already in the tailwind.config.js, if not you need to add it.
4. Check the colors if it is already in the tailwind.config.js, if not you need to add it.
5. Then whenever you do page development, always run "npm start" in the command/terminal. It will generate/update the style.css file whenever you add tailwind classes.
6. If needed that you need to add custom css, please add it on the css folder under assets.
7. All images should be added inside img folder under assets.
